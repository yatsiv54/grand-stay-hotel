# Grand Stay Hotel 🏨

![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=flat-square&logo=Flutter&logoColor=white)

Experience luxury at your fingertips with Grand Stay Hotel. Seamlessly book premium rooms, schedule exclusive events, and manage your dining experiences in one place.

---

## 🚀 Status
- **Google Play Store:** ✅ Verified & Published
- **Current Version:** 1.0.0
- **Build:** Flutter Stable Channel

## 📖 Overview
A specialized mobile solution developed for a **Gambling Affiliate Program**, designed to engage high-value users through a sophisticated hotel management interface, interactive maps, and a seamless booking ecosystem.

## ✨ Key Features
* **Smart Booking System:** Effortlessly browse and book rooms, events, and dining sessions.
* **QR-Powered Access:** Instant booking confirmation and check-ins via integrated QR code technology.
* **Interactive Hotel Map:** Advanced visual navigation to help users explore hotel facilities and event locations.
* **Personalized Schedule:** A dedicated dashboard for users to track their upcoming bookings and activities.
